🌀 What is pkgeist?

pkgeist is like if Homebrew and Pacman got possessed by a sarcastic AI ghost.

  • 💀 Beautiful CLI: geist install spooky-package
  • 🧠 AI-assisted search & resolution
  • 🌐 User-made taps (ghost towns full of packages!)
  • 🦀 Rust where it counts, Python where it's friendly

🎃 How pkgeist Works

Discover how pkgeist makes package management fun and spooky!

  • 1. Use geist tap to add your ghost town
  • 2. Search with geist search for that perfect package
  • 3. Install with geist install and get spooky packages
